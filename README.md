# Real-Mart-Analysis
My project, Real Mart Analysis, is a comprehensive business analytics dashboard built in Power BI using raw retail sales data. The goal was to transform a large dataset into meaningful insights that highlight sales performance, customer behavior, and operational efficiency across multiple categories and regions in India..
# Real Mart Analysis – Power BI Dashboard

## 📌 Overview
This project transforms raw retail sales data into an interactive Power BI dashboard, providing insights into sales performance, customer behavior, and regional trends.

## ⚙️ Key Metrics
- Orders: 28.76K  
- Total Sales: ₹83.51M  
- Net Quantity: 39K  
- Returns: 317  

## 📊 Dashboard Features
- Sales trends by year/month  
- Category & sub-category breakdown  
- Payment mode distribution  
- Returns analysis  
- Regional sales map  

## 🛠️ Tools Used
- Microsoft Power BI  
- Excel (raw dataset preparation)  

## 💡 Learning Outcomes
- Data cleaning & preparation  
- Building interactive dashboards  
- Applying visualization best practices
